# Питомник “Аристократ”

Сайт “Аристократ”— это элитная платформа, демонстрирующая кошек как настоящих чемпионов с безупречной родословной. Задача сайта — привлечь серьёзных покупателей, заинтересованных в выставках и разведении, и обеспечить удобный процесс выбора и резервирования котёнка через личный кабинет и систему анкетирования.

## Запуск для разработки (Laravel Sail)

Все команды Laravel/Node для разработки выполняйте через Sail, как в примерах ниже.

Быстрый путь:
- `composer setup`
- `./vendor/bin/sail up -d`

Пошагово:
1. Подготовить env:
   - `composer install`
2. Подготовить env:
   - `cp .env.example .env`
3. Настройте файл `.env`, указав необходимые параметры (например, настройки базы данных).
4. Запустить окружение:
   - `./vendor/bin/sail up -d`
5. Установить PHP-зависимости:
   - `./vendor/bin/sail composer install`
6. Сгенерировать ключ приложения:
   - `./vendor/bin/sail artisan key:generate`
7. Выполнить миграции:
   - `./vendor/bin/sail artisan migrate`
8. Установить JS-зависимости:
   - `./vendor/bin/sail npm install`
9. Создать ссылку на storage:
   - `./vendor/bin/sail artisan storage:link`
10. Запустить Vite:
    - `./vendor/bin/sail npm run dev`

## Команда create-admin

Команда создаёт админ-аккаунт.

Запуск:
- `./vendor/bin/sail artisan create-admin`

Поля и значения по умолчанию (если ничего не вводить):
- Email администратора: `admin@example.com`
- Пароль администратора: `admin`
- ФИО администратора: `Admin`
- Телефон администратора: `+79124567890`
- Город администратора: `Москва`
- Опыт с животными: `Нет опыта`
- Условия проживания: `Хорошие`
